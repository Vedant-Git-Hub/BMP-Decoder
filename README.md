Decodes the BMP monochrome file and displays it on the console.



How to Run:



1. Open the image you want to print on console using paint.
2. Save as the image as BMP monochrome. Important to select monochrome as this code is written for monochrome images only.
3. Copy the image path and update the macro "IMAGE\_PATH" with the copied path.
4. "DISP\_BUFF\_SIZE" macro can be updated depending on what size of buffer you want, not necessary to update in most of the cases.
5. Build the code and hit the RUN button. The image will be printed on the console.
